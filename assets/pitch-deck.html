HOW IT WORKS

From JD to shortlist in under 10 minutes

1

JD Submitted

Recruiter pastes a job description. Orchestrator parses it — extracts skills, experience, location, domain — and activates the swarm.

2

Multi-Source Sourcing

Sourcer Agent searches LinkedIn, GitHub, Naukri, and internal databases simultaneously. Deduplicates profiles. Returns confidence-ranked list.

3

Semantic Matching

Matcher Agent embeds JD + each profile using BGE-M3. Scores on 5 dimensions: skills, experience, location, domain, and transferable skills. Full breakdown per candidate.

4

Voice Screening

Screener Agent calls top candidates. Natural conversation in Hinglish — adaptive, contextual, fair. Evaluates communication, confidence, and cultural fit. Produces a report with transcript.

5

Auto-Scheduling

Scheduler Agent checks recruiter calendar, proposes slots, negotiates with candidates, sends calendar invites and reminders. No email tennis.

6

Human Review

Recruiter reviews the final shortlist with scoring breakdown, voice assessment report, and call transcripts. One-click feedback loop to improve future matching.

Communication protocol: All agents communicate via async message bus with standard envelope format — sender, recipient, type, payload, timestamp, message ID. Every message is logged. Every decision is auditable.
